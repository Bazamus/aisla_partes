# Aisla Partes - Demo

Aplicación demo de gestión de partes de trabajo.

## Tecnologías

- React + Vite
- TailwindCSS
- Supabase (PostgreSQL)
- PWA

## Configuración

1. Crear proyecto en Supabase
2. Ejecutar `demo_database_setup.sql` en el SQL Editor
3. Copiar `.env.example` a `.env` y rellenar credenciales
4. `npm install && npm run dev`
