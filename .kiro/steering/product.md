# Product Overview

**Partes Wind** is a responsive web application for generating and managing work orders (partes de trabajo) in Spanish.

## Core Features
- Work order creation and management for employees and suppliers
- Multi-role user system (Admin, Employee, Supplier)
- PDF and Excel export capabilities
- Email and WhatsApp integration
- Real-time status tracking
- Audit logging
- Price management and supplier integration
- Mobile-responsive design

## User Types
- **Administrators**: Full system access, user management, role assignment
- **Employees**: Create and manage employee work orders, view assigned projects
- **Suppliers**: Create supplier work orders, manage assigned projects

## Key Business Logic
- Role-based access control with granular permissions
- Work order workflow management
- Project assignment and tracking
- Cost calculation and reporting
- Document generation and sharing
